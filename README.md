# 🏪 The OrderNow – Ecommerce Website

A modern full-stack Ecommerce platform built with **React, Vite, Django REST Framework, and PostgreSQL**. The application enables users to browse products, manage their shopping basket, authenticate securely, and place orders seamlessly.

---

## 🚀 Features

### 👤 Customer Features

* User Registration & Login
* JWT Authentication
* Product Browsing
* Product Search & Filtering
* Category-wise Products
* 🧺 Shopping Basket
* Add to Cart / Remove from Cart
* Order Placement
* User Profile Management
* Responsive Design

### 👨‍💼 Admin Features

* Product Management
* Category Management
* Order Management
* User Management
* Inventory Tracking
* Django Admin Dashboard

## ⚙️ Installation

### Clone Repository

```bash
git clone <repository-url>
cd The-OrderNow-Ecommerce-Website
```

### Backend Setup

```bash
cd backend

python -m venv venv

# Windows
venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py createsuperuser

python manage.py runserver
```

Backend URL:

```text
http://127.0.0.1:8000
```

---

### Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

Frontend URL:

```text
http://localhost:5173
```
